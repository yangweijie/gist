## 6. 优先级分类

### 高优先级 (用户直接可见)
1. 页面标题和主要按钮
2. 导航菜单
3. 表单标签
4. 成功/错误消息

### 中优先级 (交互相关)
1. 提示文本
2. 占位符文本
3. 帮助文本
4. 验证消息

### 低优先级 (开发相关)
1. 日志消息
2. 调试信息
3. 配置项描述

## 7. 建议的语言文件结构

```
resources/lang/
├── zh/
│   ├── common.php          # 通用文本
│   ├── auth.php           # 认证相关
│   ├── gist.php           # Gist 相关
│   ├── tag.php            # 标签相关
│   ├── dashboard.php      # 控制台相关
│   ├── php-runner.php     # PHP 运行器相关
│   └── validation.php     # 验证消息
└── en/
    ├── common.php
    ├── auth.php
    ├── gist.php
    ├── tag.php
    ├── dashboard.php
    ├── php-runner.php
    └── validation.php
```
